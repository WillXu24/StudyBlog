故障表现：

- 网络时断时连及其不稳定
- 掉包严重

后台日志显示

```
kernel: mtk_esw: Link Status Changed - Port WAN Link Down
detect_link: WAN port link down detected!
kernel: mtk_esw: Link Status Changed - Port WAN Link Up
detect_link: WAN port link down restored!
```

解决方案：

- 检查你的网线插口，拔出后重新插入
- 调整网线的缠绕位置和摆放位置
- 注意平时不要碰触到网线
- 以上都做到检查问题是否解决
